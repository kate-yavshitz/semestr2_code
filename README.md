# Brouhaha Audio Quality Analysis

Анализ качества аудиозаписей с использованием модели [pyannote/brouhaha](https://huggingface.co/pyannote/brouhaha) для оценки показателей VAD, SNR и C50.

## Описание

Проект предназначен для автоматического анализа аудиозаписей (формат WAV) с помощью модели Brouhaha. Для каждого аудиофайла вычисляются:

- **VAD (Voice Activity Detection)** — обнаружение голосовой активности (0 или 1)
- **SNR (Signal-to-Noise Ratio)** — отношение сигнал/шум
- **C50** — чёткость речи (уровень реверберации)

Результаты сохраняются в виде:
- Подробных CSV-файлов для каждого аудиофайла (`имя_файла_brouhaha.csv`)
- Сводной статистики по всем файлам (`summary_statistics.csv`)
- Общего текстового отчёта (`results.txt`, опционально)

## Инструкция
1.	Перейти в папку проекта, открыть командную строку, установить проект.
Используйте файл helper.txt из папки notebook и по шагам выполните установки необходимой версии python (версия 3.8) и важных зависимостей.

Создайте виртуальное окружение:
venv38\Scripts\activate

2. Установить все зависимости из requirements.txt
pip install -r requirements.txt

4.	Получить токен на портале https://huggingface.co/ и вписать его в файл .env в формате:
HF_TOKEN=новый_токен и вставить его в .env brouhaha.ipynb и run_brouhaha.py на место слова secret

5. Запуск программы для анализа папки с аудиофайлами
3.1) С помощью программного обеспечения VS Code открыть папку проекта.
3.2) Открыть файл: notebooks/brouhaha.ipynb. 
3.3) Выбрать интерпретатор с виртуальным окружением (нажать в правом нижнем и выбрать venv).
3.4) Выполнить ячейки файла brouhaha.ipynb по порядку

6.	Результаты работы
После обработки в папке с аудиофайлами появятся:
4.1 Для каждого аудиофайла:
имя_файла_brouhaha.csv — покадровые данные с колонками:

timestamp — время в секундах
is_vad — наличие речи (1/0)
snr — отношение сигнал/шум
c50 — чёткость речи

4.2 Сводная статистика:
summary_statistics.csv — общая статистика по всем файлам:
file_name — имя файла
snr_mean, snr_median, snr_std, snr_min, snr_max
c50_mean, c50_median, c50_std, c50_min, c50_max
vad_speech_ratio — доля времени с речью

4.3 Опционально:
results.txt — полный текстовый отчёт (если выбран при запуске)
